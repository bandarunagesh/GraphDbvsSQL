# GraphDbvsSQL

**Executive Demo: Graph DB Ontology Layer vs Traditional SQL Agents for Agentic AI in Clinical Trials Operations**

This project contains:
- **Slide Deck** - 11-slide executive presentation proposing Graph DB as the Semantic/Ontology Layer
- **Live Demo App** - Side-by-side comparison of SQL Agents vs Graph DB Ontology-Aware Agents across 4 scenarios:
  1. Patient Eligibility (Context Stitching)
  2. Safety Signal Detection (Context Stitching)
  3. Site Performance (Context Stitching)
  4. Root Cause Analysis (Multi-Hop Graph Traversal)

Built with React + Vite, TypeScript, TailwindCSS
